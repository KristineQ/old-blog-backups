本文是写自己尝试从零开始搭一个匿名聊天室的学习、经验和踩坑。

搭这个的原因，是因为之前看《 无头骑士异闻录（DRRR）》动漫，里面有一个叫 DOLLARS 的匿名聊天室，非常喜欢，也有人已经做出来还原度非常高的的网页版 DOLLARS 聊天室，并且持续维护在 drrr.com 。

也许之前学做网站的理由也是想亲手做一个那样的聊天室，趁现在还有时间，或许能尝试着完成一下心愿。
