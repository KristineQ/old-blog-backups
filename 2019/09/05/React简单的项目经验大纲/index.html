React 基础的一些理解

  • JSX 就是createELement()的简便写法，就相当于appendChild()，里面的相当于 props，babel

  • bind(this,...props)组件作用域的不同 调用函数的作用域和原函数的作用域不同导致的找不到相应变量的错误相似，可以在没有太多参数需要传给 constructor 的情况下直接把 state 提到公共类,然后 bind 方法到当前环境，也是可以直接利用箭头函数写在当前环境
